# EcoTextile Marketplace

EcoTextile Marketplace is a full-stack web application designed to promote sustainable and eco-friendly textile trading. The platform connects buyers and sellers, enabling them to list, discover, and purchase environmentally responsible textile products while supporting a circular economy.

---

## Features

- User authentication (Sign up / Login)
- Browse eco-friendly textile products
- Search, filter, and sort products
- Product listing management for sellers
- Cart and order management
- Secure backend APIs
- Responsive and user-friendly interface

---

## Tech Stack

- **Frontend:** React / Next.js
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Styling:** CSS / Tailwind CSS
- **Authentication:** JWT
- **Deployment:** Vercel (Frontend), Render/Heroku (Backend)
